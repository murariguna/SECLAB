What SQL injection does

A vulnerable query concatenates user input directly into SQL. Injecting ' OR '1'='1 makes the condition always true, exposing every row.

A parameterised query treats input as pure data — the database never interprets it as SQL — making injection impossible.

Safe vs vulnerable

❌ Vulnerable

"SELECT * FROM users WHERE username = '" + username + "'"

✅ Safe (parameterised)

"SELECT * FROM users WHERE username = ?" , (username,)
